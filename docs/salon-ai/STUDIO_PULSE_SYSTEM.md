# Studio Pulse System

## Module

- Name: `Studio Pulse`
- Route: `/studio-pulse`
- Purpose: internal salon intelligence for incidents, feedback, suggestions, and AI observations

## Positioning

Studio Pulse reframes SendMe Studio from a messaging layer into an AI Salon Operating System.

It centralizes:

- incidencias
- observaciones
- sugerencias
- reclamos
- mejoras
- oportunidades
- alertas IA

## Core Sections

### 1. Incidents

For operational problems such as:

- atraso estilista
- mala atención
- error técnico
- limpieza
- producto agotado
- cliente molesto

Visual direction:

- soft red / soft amber tension
- calm escalation, not alarmist enterprise UI

### 2. Client Feedback

Captures what clients explicitly said, then lets AI detect:

- emociones
- patrones
- riesgos
- oportunidades

Examples:

- “muy feliz con el resultado”
- “esperó demasiado”
- “quiere más privacidad”
- “amó el masaje capilar”

### 3. Studio Suggestions

Internal suggestions from the team to build premium culture.

Examples:

- cambiar iluminación
- agregar café premium
- nueva playlist
- reorganizar recepción
- mejorar protocolo WhatsApp

### 4. AI Observations

AI analyzes:

- tiempos
- conversaciones
- reservas
- cancelaciones
- palabras frecuentes
- clientes VIP

And generates observations such as:

- clientes Balayage VIP responden mejor después de las 18:00 hrs
- las clientas premium mencionan “espera” 28% más esta semana
- existe alta afinidad hacia rituales de hidratación profunda

## Features Included

- crear nota
- crear incidencia
- adjuntar foto
- modo silencioso / anónimo
- IA resume automáticamente
- IA recomienda acción
- estado de resolución
- timeline elegante con impacto y ownership

## Visual Direction

Not Jira, not Trello, not a classic CRM.

Designed to feel like:

- Apple Notes premium
- Linear soft
- Notion AI luxury
- beauty operations OS

Visual language:

- calm glass surfaces
- pearl / lavender gradients
- soft editorial spacing
- delicate borders
- invisible internal scroll
- premium intelligence presentation
