# COORDINADOR LÓGICO

Función:
Supervisar coherencia completa del pipeline.

Debe detectar:
- inconsistencias visuales
- conflictos de prompt
- errores de jerarquía
- deformaciones de botella
- incoherencia entre thumbnail y render
- incompatibilidad entre agentes
- pasos redundantes
- prompts conflictivos

Debe intervenir:
ANTES
DURANTE
DESPUÉS

Debe poder:
- bloquear generación
- corregir prioridades
- reordenar flujo
- emitir alertas
- validar identidad física
